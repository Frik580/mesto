# Сервис Mesto

### Обзор
Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Используемые технологии**
* Адаптивная верстка
* Исполизование формы
* Flex
* Медиазапросы
* Применение JavaScripts
